# Voice Text Enhancer - 5 分钟快速安装

## 🚀 第一步：安装应用（1 分钟）

```bash
# 复制应用到 Applications 文件夹
cp -r dist/VoiceTextEnhancer.app /Applications/
```

---

## ⚙️ 第二步：配置 API Key（1 分钟）

```bash
# 编辑环境变量文件
nano ~/.voice-text-enhancer/.env
```

添加你的 API Key：
```
DEEPSEEK_API_KEY=sk-你的密钥
```

**保存：** Ctrl+X → Y → Enter

**获取 API Key：** https://platform.deepseek.com/api_keys

---

## 🔐 第三步：授予权限（2 分钟）

### 1. 辅助功能权限
- 系统设置 > 隐私与安全性 > **辅助功能**
- 点击 **+** 添加 **VoiceTextEnhancer**
- 确保开关打开 ✓

### 2. 输入监控权限
- 系统设置 > 隐私与安全性 > **输入监控**
- 点击 **+** 添加 **VoiceTextEnhancer**
- 确保开关打开 ✓

---

## 🎉 第四步：启动使用（1 分钟）

```bash
# 启动应用
open /Applications/VoiceTextEnhancer.app
```

或在 Spotlight (Cmd+空格) 中搜索"VoiceTextEnhancer"启动。

---

## ✨ 使用方法

```
1. 在任意输入框输入文本
2. 轻按 右 Option 键 ⌥
3. 自动优化完成 ✓
```

**示例：**

输入：`今天要做三件事第一写文档第二开会第三写代码`

按右 Option 后自动变成：
```
今天要做三件事：

1. 写文档
2. 开会
3. 写代码
```

## 🔧 常见问题

### Q: 应用打不开？
```bash
# 移除安全限制
xattr -cr /Applications/VoiceTextEnhancer.app
```

### Q: 快捷键没反应？
1. 检查应用是否运行：`ps aux | grep VoiceTextEnhancer`
2. 检查权限是否授予（看上面第三步）

### Q: 如何退出应用？
```bash
pkill -f VoiceTextEnhancer
```

### Q: 查看日志？
```bash
tail -f ~/.voice-text-enhancer/app.log
```
