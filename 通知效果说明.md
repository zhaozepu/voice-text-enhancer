# 🎨 彩色动效通知说明

## ✨ 新功能

现在使用**漂亮的彩色浮层通知**，显示在**屏幕右下角**！

---

## 🎯 通知类型

### 1. 处理中通知（蓝色 + 旋转动画）

**颜色：** 蓝色 (#4A90E2)  
**图标：** ⟳ 旋转图标  
**位置：** 右下角浮层  
**效果：** 淡入动画 + 旋转动画  

```
┌─────────────────────────────┐
│  ⟳  正在处理文本，请稍候...    │
└─────────────────────────────┘
```

**触发时机：** 按快捷键后立即显示


### 2. 成功通知（绿色）

**颜色：** 绿色 (#52C41A)  
**图标：** ✓ 对勾  
**位置：** 右下角浮层  
**效果：** 淡入淡出动画  
**持续：** 2 秒自动消失  

```
┌─────────────────────────────┐
│  ✓  文本处理完成              │
└─────────────────────────────┘
```

**触发时机：** 文本处理完成并替换后


### 3. 错误通知（红色）

**颜色：** 红色 (#F5222D)  
**图标：** ✗ 叉号  
**位置：** 右下角浮层  
**效果：** 淡入淡出动画  
**持续：** 3 秒自动消失  

```
┌─────────────────────────────┐
│  ✗  处理失败：网络连接错误    │
└─────────────────────────────┘
```

**触发时机：** 发生错误时（如 API 失败）


### 4. 信息通知（浅蓝色）

**颜色：** 浅蓝 (#1890FF)  
**图标：** ℹ 信息图标  
**位置：** 右下角浮层  
**效果：** 淡入淡出动画  
**持续：** 2-3 秒自动消失  

```
┌─────────────────────────────┐
│  ℹ  工具已启动                │
│     轻按右 Option 即可处理    │
└─────────────────────────────┘
```

**触发时机：** 程序启动时

---

## 🎬 完整工作流程

```
用户按快捷键
    ↓
┌─────────────────────────┐
│ ⟳ 正在处理文本，请稍候... │  ← 蓝色，旋转动画
└─────────────────────────┘
    ↓
  等待 API 处理（2-5秒）
    ↓
自动关闭 → 淡出
    ↓
┌─────────────────────────┐
│ ✓ 文本处理完成           │  ← 绿色，2秒后自动消失
└─────────────────────────┘
```

## 🎨 设计特点

### 视觉效果
✅ **彩色主题** - 蓝/绿/红/浅蓝四种颜色  
✅ **半透明** - 95% 不透明度，优雅不突兀  
✅ **圆角设计** - 现代化界面  
✅ **无边框** - 纯浮层效果  

### 动画效果
✅ **淡入动画** - 0 → 95% 渐变显示  
✅ **淡出动画** - 95% → 0 渐变消失  
✅ **旋转动画** - 处理中图标持续旋转  
✅ **自动消失** - 成功/错误通知自动关闭  

### 交互体验
✅ **右下角定位** - 不遮挡工作区  
✅ **置顶显示** - 始终在最前面  
✅ **自动管理** - 无需手动关闭  
✅ **流畅切换** - 通知之间平滑过渡

## ⚙️ 配置选项

在 `config.yaml` 中可以开关通知：

```yaml
notifications:
  enabled: true          # 总开关
  show_processing: true  # 显示处理中通知
  show_completion: true  # 显示完成通知
  show_errors: true      # 显示错误通知
```

## 💡 使用示例

### 示例 1: 正常工作流

```
1. 输入："今天要做三件事..."
2. 按右 Option
3. 看到蓝色浮层："⟳ 正在处理文本..."（旋转动画）
4. 3 秒后自动变绿色："✓ 文本处理完成"
5. 2 秒后自动消失
```

### 示例 2: 空文本

```
1. 输入框为空
2. 按右 Option
3. 红色浮层："✗ 请先在输入框中输入文本"
4. 3 秒后自动消失
```

### 示例 3: 网络错误

```
1. 输入文本并按快捷键
2. 蓝色浮层："⟳ 正在处理文本..."
3. API 调用失败
4. 红色浮层："✗ 处理失败：网络连接错误"
5. 3 秒后自动消失
```

## 🎁 额外功能

### 智能切换
- 处理中通知会自动被成功/错误通知替换
- 无需手动关闭，流畅过渡

### 防重复
- 同时只显示一个通知
- 新通知会替换旧通知

### 线程安全
- 异步调用不阻塞主程序
- 动画在独立线程运行
