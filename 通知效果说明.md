# 🎨 桌面浮层通知说明

## ✨ 新功能

现在使用 **pywebview + HTML/CSS** 实现的**彩色浮层通知**，显示在**屏幕右下角**！

**技术实现：**
- 使用 pywebview 创建无边框透明窗口
- HTML/CSS 渲染炫酷渐变背景和动画效果
- 支持自动关闭、置顶显示、流畅过渡

---

## 🎯 通知类型

### 1. 处理中通知（紫色渐变 + 旋转动画）

**颜色：** 紫色渐变 (#667eea → #764ba2)  
**图标：** ⟳ 旋转图标（无限旋转）  
**位置：** 右下角浮层  
**效果：** 滑入动画 + 旋转动画  

```
┌─────────────────────────────┐
│  ⟳  正在处理文本，请稍候...    │
└─────────────────────────────┘
```

**触发时机：** 按快捷键后立即显示


### 2. 成功通知（绿色渐变）

**颜色：** 绿色渐变 (#52c234 → #43a047)  
**图标：** ✓ 对勾（脉冲动画）  
**位置：** 右下角浮层  
**效果：** 滑入滑出动画  
**持续：** 2 秒自动消失  

```
┌─────────────────────────────┐
│  ✓  文本处理完成              │
└─────────────────────────────┘
```

**触发时机：** 文本处理完成并替换后


### 3. 错误通知（红色渐变）

**颜色：** 红色渐变 (#ff6b6b → #ee5a6f)  
**图标：** ✗ 叉号（脉冲动画）  
**位置：** 右下角浮层  
**效果：** 滑入滑出动画  
**持续：** 3 秒自动消失  

```
┌─────────────────────────────┐
│  ✗  处理失败：网络连接错误    │
└─────────────────────────────┘
```

**触发时机：** 发生错误时（如 API 失败）


### 4. 信息通知（蓝色渐变）

**颜色：** 蓝色渐变 (#4facfe → #00f2fe)  
**图标：** ℹ️ 信息图标  
**位置：** 右下角浮层  
**效果：** 滑入滑出动画  
**持续：** 2-3 秒自动消失  

```
┌─────────────────────────────┐
│  ℹ  工具已启动                │
│     轻按右 Option 即可处理    │
└─────────────────────────────┘
```

**触发时机：** 程序启动时

```
用户按快捷键
    ↓
┌─────────────────────────┐

    ↓
┌─────────────────────────┐

```

## 🎨 设计特点

### 视觉效果
✅ **渐变背景** - 紫/绿/红/蓝四种精心设计的渐变色  
✅ **玻璃模糊** - backdrop-filter 毛玻璃效果  
✅ **圆角设计** - 12px 圆角，现代化界面  
✅ **无边框** - frameless=True，纯浮层效果  
✅ **透明窗口** - 支持透明背景  

### 动画效果
✅ **滑入动画** - 从下方滑入，0.3s 缓动  
✅ **滑出动画** - 向下方滑出，0.3s 缓动  
✅ **旋转动画** - 处理中图标 360° 无限旋转  
✅ **脉冲动画** - 成功/错误图标缩放脉冲  
✅ **自动消失** - 成功/错误通知自动关闭  

### 交互体验
✅ **右下角定位** - 不遮挡工作区  
✅ **置顶显示** - 始终在最前面  
✅ **自动管理** - 无需手动关闭  
✅ **流畅切换** - 通知之间平滑过渡

## ⚙️ 配置选项

在 `config.yaml` 中可以开关通知：

```yaml
notifications:
  enabled: true          # 总开关
  show_processing: true  # 显示处理中通知
  show_completion: true  # 显示完成通知
  show_errors: true      # 显示错误通知
```

## 💡 使用示例

### 示例 1: 正常工作流

```
1. 输入："今天要做三件事..."
2. 按右 Option
3. 看到紫色渐变浮层："⟳ 正在处理文本..."（旋转动画）
4. 3 秒后滑出，自动变绿色渐变："✓ 文本处理完成"（脉冲动画）
5. 2 秒后滑出消失
```

### 示例 2: 空文本

```
1. 输入框为空
2. 按右 Option
3. 红色渐变浮层："✗ 请先在输入框中输入文本"（脉冲动画）
4. 3 秒后滑出消失
```

### 示例 3: 网络错误

```
1. 输入文本并按快捷键
2. 紫色渐变浮层："⟳ 正在处理文本..."（旋转动画）
3. API 调用失败
4. 红色渐变浮层："✗ 处理失败：网络连接错误"（脉冲动画）
5. 3 秒后滑出消失
```

## 🎁 额外功能

- 新通知会替换旧通知

- 异步调用不阻塞主程序
- 动画在独立线程运行
