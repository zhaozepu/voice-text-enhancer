# 📝 更新日志

## v1.1.0 - 快捷键与 Typeless 智能结构化 (2026-05-27)

### ✨ 新功能

#### ⌨️ 简化快捷键
- 从组合键 (Cmd+Shift+P) 改为单键触发
- 默认快捷键：**右 Option 键**
- 支持单次轻按，防止长按误触发
- 可选：left_option、right_cmd

#### 🤖 Typeless 智能结构化
- 新增 "typeless" prompt 模式（默认启用）
- 自动识别文本类型并智能结构化
- 支持 Markdown 格式输出：
  - 多个要点 → 有序/无序列表
  - 明确主题 → 标题 + 列表
  - 关键信息 → 粗体突出
- 纠正语法错误和错别字
- 优化表达，保持简洁专业

#### 🎯 自动全选功能
- 无需手动选中文本
- 按快捷键自动全选输入框内容 (Cmd+A)
- 自动复制、处理、替换

#### 🎨 增强通知系统
- macOS 原生通知 + 终端彩色输出
- 4 种通知类型：处理中（蓝色动画）、成功（绿色）、错误（红色）、信息（青色）
- 终端旋转动画指示处理进度
- 音效提示（错误时）

### 🔧 技术改进

#### 快捷键监听优化
- `src/core/hotkey_listener.py` 支持单键模式
- 添加按键持续时间检测（< 0.5s）
- 防止长按触发多次

#### 剪贴板时序优化
- 增加剪贴板写入延迟至 0.5s
- 添加剪贴板内容验证机制
- 修复粘贴内容不正确的问题

#### Prompt 系统升级
- 在 `config.yaml` 中新增 typeless 模式
- 支持多种 prompt 切换
- 默认使用智能结构化模式

### 🐛 问题修复
- 修复剪贴板内容被替换的问题
- 修复快捷键冲突问题
- 修复文本获取时序问题
- 解决"请先选中文本"误报

### 📚 文档更新
- 新增 TYPELESS_EXAMPLES.md - 7 个智能结构化示例
- 更新使用文档，说明新快捷键
- 添加 APP 使用指南
- 更新通知效果说明
