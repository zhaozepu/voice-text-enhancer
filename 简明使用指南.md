# Voice Text Enhancer - 极简使用指南

## 🎯 一句话说明

**语音转文字后，按一下右 Option 键，自动优化并结构化文本。**

---

## 🚀 快速启动

```bash
cd /Users/tal/Code/voice-text-enhancer
./run.sh
```

---

## 💡 使用方法

### 三步走

```
1️⃣ 输入文本（豆包语音 or 手动输入）
          ↓
2️⃣ 轻按 右 Option 键 ⌥
          ↓
3️⃣ 自动优化完成 ✓
```

### 实际例子

**输入：**
```
今天要做三件事第一写文档第二开会第三写代码
```

**按右 Option 后：**
```
今天要做三件事：

1. 写文档
2. 开会  
3. 写代码
```

---

## ⚙️ 首次配置

### 1. 安装依赖
```bash
bash scripts/install.sh
```

### 2. 配置 API Key
编辑 `.env` 文件：
```
DEEPSEEK_API_KEY=sk-你的密钥
```
获取地址：https://platform.deepseek.com/api_keys

### 3. 授予权限
- 系统设置 > 隐私与安全性 > **辅助功能** ✓ 添加"终端"
- 系统设置 > 隐私与安全性 > **输入监控** ✓ 添加"终端"

**必须重启终端才能生效！**

## 🔧 常用操作

### 更换快捷键
编辑 `config.yaml`：
```yaml
hotkey:
  trigger: "right_option"  # 改成你想要的
```

### 切换模式
编辑 `config.yaml`：
```yaml
active_prompt: "typeless"  # 智能结构化（默认）
# active_prompt: "correct"   # 仅纠错
# active_prompt: "optimize"  # 纠错+优化
```

---

## 🐛 遇到问题？

### 快捷键没反应
1. 检查权限是否授予（看上面"授予权限"）
2. 完全重启终端

### 文本没被替换
1. 光标必须在输入框内
2. 输入框必须有文本

### 查看日志
```bash
tail -f ~/.voice-text-enhancer/app.log
```
